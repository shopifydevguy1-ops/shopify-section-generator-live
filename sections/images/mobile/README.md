# Mobile Preview Images

Place mobile-specific preview images for your sections here.

## Naming Convention

Mobile images should be named to match their corresponding section file:
- Section file: `sg-hero-1.liquid` → Mobile image: `sg-hero-1.png`
- Section file: `allbirds-banner.liquid` → Mobile image: `allbirds-banner.png`

## Supported Formats

- PNG (recommended)
- JPG/JPEG
- WebP
- SVG

## Example

If you have a section file at `sections/sg-hero-1.liquid`, you can add:
- Desktop preview: `sections/images/sg-hero-1.png`
- Mobile preview: `sections/images/mobile/sg-hero-1.png`

The mobile image will automatically be displayed in the mobile device mockup when available. If no mobile image is provided, the desktop preview image will be used for both laptop and mobile mockups.
