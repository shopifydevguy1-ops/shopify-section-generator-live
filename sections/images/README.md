# Section Preview Images

Place preview images for your sections here.

## Naming Convention

Images should be named to match their corresponding section file:
- Section file: `sg-hero-1.liquid` → Image: `sg-hero-1.png`
- Section file: `allbirds-banner.liquid` → Image: `allbirds-banner.png`

## Supported Formats

- PNG (recommended)
- JPG/JPEG
- WebP
- SVG

## Example

If you have a section file at `sections/sg-hero-1.liquid`, you can add a preview image at:
`sections/images/sg-hero-1.png`

The preview image will automatically be displayed in the generator interface when searching for sections.
